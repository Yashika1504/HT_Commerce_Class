# 🚀 HT Commerce Classes

## 📌 About

HT Commerce Classes is a Django-based web application developed to digitize and streamline student inquiries for a commerce coaching institute. The platform provides a clean and responsive user interface, allowing students to easily connect with the institute.

It includes features like an inquiry form and WhatsApp Cloud API integration, enabling real-time communication between students and the institute. This system helps traditional offline coaching institutes establish a strong digital presence and improve interaction efficiency.

---

## ✨ Features

* 📩 Student Inquiry Form
* 📱 WhatsApp Cloud API Integration
* 🎨 Responsive UI (Mobile + Desktop Friendly)
* ⚡ Fast and Lightweight Django Backend
* 🧑‍🏫 Designed for Coaching Institutes
* 🔒 Secure Data Handling

---

## 🛠️ Tech Stack

* **Backend:** Django (Python)
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite
* **API Integration:** WhatsApp Cloud API

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Yashika1504/HT_Commerce_Class.git
cd HT_Commerce_Class
```

### 2️⃣ Create Virtual Environment

```bash
python -m venv venv
venv\Scripts\activate
```

### 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Server

```bash
python manage.py runserver
```

👉 Open browser: http://127.0.0.1:8000/
